# Burhan Ahmed - Production Portfolio Website

A minimal, animated, and production-ready portfolio website built with Flask and SQLite.

## Features

### 🎨 Frontend
- Minimal and clean design
- Smooth animations (fade, slide, stagger, magnetic buttons)
- Dark/Light mode toggle
- Responsive mobile menu with hamburger animation
- Project showcase with featured projects
- Interactive project panels with detailed sections
- Image slider modal for project galleries
- Newsletter subscription with AJAX
- Scroll progress bar
- Loading animations
- Typing effect hero section
- Count-up statistics
- Scroll-based reveal animations

### 🔧 Backend
- Flask with modular architecture
- SQLite database with SQLAlchemy ORM
- Flask-Admin for content management
- Flask-Login for authentication
- Flask-Mail for email notifications
- Async email sending with threading
- Image upload with validation
- CSRF protection
- Secure file handling

### 📧 Email System
- Automated project notifications to subscribers
- HTML email templates with inline CSS
- Unsubscribe functionality with secure tokens
- Welcome emails for new subscribers
- Async sending to prevent blocking

## Tech Stack

- **Backend**: Flask, SQLAlchemy, Flask-Admin, Flask-Login, Flask-Mail
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Database**: SQLite
- **Security**: Werkzeug, python-dotenv
- **Email**: SMTP with threading
